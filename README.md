# Help Desk Ticketing System

## Requirements
- Java 21
- Maven

## Running the Application

1. Open the project in NetBeans.
2. Allow Maven to download dependencies.
3. Click Run.

The embedded Derby database is automatically created on first launch.
Default tables and required data are initialized automatically.
